#  SCT_WD_4 To-Do Web App

# Overview
A task management application that allows users to add, edit, delete, organize, and mark tasks as completed. Includes date and time support for tasks.

# Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark complete
- Set due date & time
- Organized task list

# Technologies Used
- HTML5
- CSS3
- JavaScript

# Learning Outcome
- CRUD operations
- DOM manipulation
- State management
